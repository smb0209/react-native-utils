# react-native-utils

A monorepo of reusable React Native utilities.

This repository contains small, focused packages designed for reuse in real-world React Native applications.

Each package is independently versioned and published to npm.

---

## Packages

### 1. Stroke SVG Text

An SVG-based outlined text component for React Native.

- Consistent stroke rendering on iOS and Android
- Built on top of `react-native-svg`
- Designed for decorative or animation-heavy UI

### 2. Keyboard Inset

Utilities for handling keyboard insets, overlays, and stable cursor behavior in React Native.

- Overlay registration for floating toolbars or accessories
- Stable undo/redo cursor handling on iOS TextInput
